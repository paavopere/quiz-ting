quiz-ting

Search for place names that match a pattern

How to search: Type some letters and wildcards. ? replaces 1 or more characters and * replaces 1 character. Optionally filter by country code (e.g., US, GB, IN).

  • l?nd?n matches London, Linden, and Letpandan, etc.
  • l*nd*n matches London and Linden but not Letpandan
  • ?oh*a matches stuff like Lohja and Limbach-Oberfrohna

For hackers, you can use regex patterns if you check the box. E.g. .+oh.a
